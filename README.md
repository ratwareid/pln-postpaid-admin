# pln-postpaid-admin
Springboot Web using JPA Postgresql and thmeleaf

Admin panel pembayaran PLN Postpaid.

Dibuat menggunakan :
- Springboot
- Hibernate jpa
- Postgres
- Thymeleaf
- Lombok
- Swagger OpenAPI

Feature :

Master Data :
- User : Administrator
- Pelanggan
- Tarif
- Level : Role akses

Transaksi :
- Penggunaan : mencatat penggunaan listrik pelanggan
- Tagihan
- Pembayaran

Laporan :
- Laporan tagihan dan pembayaran
